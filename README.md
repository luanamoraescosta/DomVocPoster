# DomVocPoster

>  This project is currently under construction.

## What is this?

This project takes a Jupyter notebook used for processing monastery data and turns it into an **interactive, browser-based notebook** — no Python installation, no terminal, no technical knowledge required.

The goal is to make the analysis accessible to anyone: historians, researchers, or curious readers who just want to explore the data without touching a single line of code.

## How does it work?

Everything runs directly in the browser using [Pyodide](https://pyodide.org), a version of Python compiled to WebAssembly. When you open the page, Python loads silently in the background and you can run the notebook cells with a single click.

## Status

The project is a work in progress. Some features may be incomplete or change without notice.
